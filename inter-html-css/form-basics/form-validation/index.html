This uses minlength, if you don't use at least 3 characters you get a prompt.

Similarly, this uses maxlength. If you try to type more than 20 characters, the browser won't let you.

You can combine minlength and maxlength. (Try typing less than 5 or more than 20 characters)

For minimum number validation (i.e. ordering at least one of something) you use min.

For max number validation (i.e. limiting the amount someone can purchase) you use max

We can use the pattern attribute for things like credit card number validation, or in this case, zipcode validation.

You may also style validations by targeting valid and invalid in your stylesheet.
